- **Persistent State Management**: Remembers last viewed campaign and preferences
- **URL-based Navigation**: Deep-linking support for sharing specific content
- **Responsive Design**: Works on desktop and mobile devices
- **Local Storage**: Client-side storage for user preferences
- **No Backend Required**: Fully static site that can be hosted anywhere
- **Browser History Integration**: Full back/forward button support

## Use Cases

Perfect for:

- Dungeon Masters wanting to share campaign content with players
- Players who want to review past sessions and track quests
- Groups looking to document their adventures
- Anyone wanting a visual, organized way to present D&D campaign information

## Campaign Structure

Each campaign can include:

- **Metadata**: Name, description, level range, party composition
- **Map Data**: Interactive maps with locations and markers
- **Session Recaps**: Detailed narrative content for each session
- **Timeline Events**: Key story moments and dates
- **Quests**: Objectives, status, and rewards
- **NPCs**: Character profiles and relationships
- **Locations**: Places of interest with descriptions
- **Factions**: Organizations and their goals

## Content Types

The tool supports campaigns with:

- Map-only content (for location reference)
- Story-only content (for recap archives)
- Combined map and story content (full campaign documentation)
